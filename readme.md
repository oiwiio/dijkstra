# Алгоритм Дейкстры — визуализация

Четыре задачи на поиск кратчайших путей в графе с интерактивной SVG-визуализацией.
посмотреть можно здесь: https://oiwiio.github.io/dijkstra/

## Задачи

| # | Задача | Суть |
|---|--------|------|
| 1 | **Восстановление пути** | Кратчайший путь S→F, восстановленный по таблице parents. |
| 2 | **Все кратчайшие пути** | Когда несколько маршрутов имеют одинаковую длину — находим все, храня список родителей. |
| 3 | **Недостижимые вершины** | Изолированные узлы остаются с cost=Infinity; в результате выводится −1. |
| 4 | **Маршрутизация в сети** | Минимальная суммарная задержка (мс) между роутерами. |

## Запуск

открыть `index.html` в браузере, или по ссылке выше

```
dijkstra.js    — алгоритмы: dijkstra, dijkstraAllPaths, reconstructPath, buildGraph
index.html  — визуализация: SVG-графы, подсветка пути, таблицы расстояний
```

## Технологии

Vanilla JS · SVG · HTML — без зависимостей.
